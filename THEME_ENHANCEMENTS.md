**CSS Variables Used:**
- `--gradient-1`: Main purple gradient

**Key Animations:**
- `fadeInUp`: Content entry animation
- `pulse`: Loading states
- Custom hover transformations

### 🎨 Customization

To adjust colors, edit `/assets/css/extended/custom.css`:

```css
:root {
    --gradient-1: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    /* Change these hex values for different colors */
}
```
